# Programaticas 2024 - Dashboard

Dashboard Streamlit para revisar coberturas de vacunas programaticas 2024.

## Instalacion

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Uso

```powershell
streamlit run streamlit_dashboard.py
```

Los archivos de entrada y salida del proyecto estan organizados en `data/` y `output/`.
